### [LabVIEW Reset Protocol 1.0](#labview-reset-protocol-10)

- Description

  This example resets all settings of Dynamixel to default values as:

  * ID = 1

  * Baudrate=57600.

- Available Dynamixel

  All series using protocol 1.0
